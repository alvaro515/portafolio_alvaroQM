Teoría y Conceptos de la Semana

1. React

React es una librería de JavaScript desarrollada por **Facebook** que se utiliza para construir interfaces de usuario, especialmente aplicaciones web interactivas. React es muy popular porque permite crear aplicaciones web dinámicas con un rendimiento excelente, gracias a su sistema de renderizado eficiente.

¿Para qué sirve React?

React es ideal para desarrollar aplicaciones **single-page** (SPA), donde el contenido de la página se actualiza dinámicamente sin necesidad de recargar toda la página. Esto mejora la experiencia del usuario, haciendo que la interacción con la web sea más fluida.

Con React, puedes crear componentes reutilizables que encapsulan tanto la **estructura** (HTML), la **lógica** (JavaScript) y el **estilo** (CSS) de las distintas partes de tu interfaz. Esto facilita la organización del código y hace que sea más fácil mantener y actualizar las aplicaciones.

¿Cómo instalar React?

Para empezar a usar React, lo primero que necesitas es instalar **Node.js** en tu computadora. Luego, puedes instalar **React** a través de **npm** (Node Package Manager), que es el gestor de dependencias más utilizado en proyectos JavaScript.

Paso a paso para instalar React:
  1. Instalar Node.js: Ve a nodejs.org y descarga la versión recomendada para tu sistema operativo. Node.js incluye npm, así que no necesitas instalarlo por separado.
  2. Crear un nuevo proyecto con React: Una vez que tengas Node.js instalado, abre tu terminal (o línea de comandos) y ejecuta el siguiente comando para crear una nueva aplicación con React:
    3. npx create-react-app mi-app

    Esto creará una nueva carpeta llamada **mi-app** con todos los archivos necesarios para empezar a trabajar en tu proyecto.

  3. Iniciar la aplicación: Entra en el directorio de tu proyecto recién creado con el siguiente comando:
    4. cd mi-app

    Luego, ejecuta este comando para iniciar la aplicación:

    npm start

    Esto abrirá tu aplicación en el navegador en http://localhost:3000 y podrás empezar a editarla.

Componentes en React

React se basa en componentes, que son bloques reutilizables de código que contienen la lógica y la vista de la aplicación. Los componentes pueden ser **funcionales** o **de clase**. Sin embargo, desde la introducción de los **hooks** en React 16.8, los componentes funcionales son los más utilizados.

Ejemplo de un componente funcional en React:

        
          import React from 'react';

          function Saludo(props) {
            return <h1>Hola, {props.nombre}</h1>
          }

          export default Saludo;

Este es un componente llamado **Saludo**, que recibe un **prop** llamado **nombre** y muestra un mensaje personalizado. Los **props** son valores que se pasan a los componentes para hacerlos dinámicos.

React - JSX

JSX es una extensión de sintaxis que permite escribir HTML dentro de JavaScript. Aunque parece HTML, JSX es en realidad JavaScript y se compila a funciones de React. Usar JSX facilita la creación de interfaces de usuario y hace que el código sea más legible.

Ejemplo de JSX:

        
          const element = <h1>¡Hola, React!</h1>
          ReactDOM.render(element, document.getElementById('root'));

2. Node.js

Node.js es un entorno de ejecución de JavaScript en el servidor. Es especialmente útil para construir aplicaciones web escalables y de alto rendimiento. Node.js permite ejecutar código JavaScript fuera del navegador, lo que lo hace ideal para aplicaciones del lado del servidor, como **APIs**.

¿Para qué sirve Node.js?

Node.js es ideal para desarrollar aplicaciones que requieren procesamiento en tiempo real, como chats, aplicaciones de colaboración en línea y servidores web. Además, es muy eficiente para manejar un gran número de conexiones simultáneas gracias a su modelo de **event loop** no bloqueante.

¿Cómo instalar Node.js?

Ve a nodejs.org y descarga la última versión estable. Luego, abre tu terminal y verifica la instalación ejecutando:

node -v

3. Vite

Vite es una herramienta moderna para desarrollo de aplicaciones JavaScript que se enfoca en ser **rápida** y **eficiente**. Vite utiliza **ESBuild** para la compilación, lo que permite tiempos de arranque y recarga de página muy rápidos en comparación con otras herramientas como Webpack.

¿Para qué sirve Vite?

Vite es excelente para proyectos de **React**, **Vue**, y **Svelte**, y proporciona una experiencia de desarrollo increíblemente rápida gracias a su sistema de **HMR (Hot Module Replacement)**, lo que significa que solo se actualizan los módulos que cambian sin recargar toda la página.

¿Cómo instalar Vite?

Para empezar a usar Vite, primero necesitas tener **Node.js** instalado. Luego, puedes crear un nuevo proyecto usando Vite con el siguiente comando:


        npm create vite@latest nombre-del-proyecto --template react

Este comando creará un proyecto de React con Vite. Luego, navega a tu carpeta de proyecto:

cd nombre-del-proyecto

Y finalmente, instala las dependencias y ejecuta el servidor de desarrollo:

npm install
npm run dev

Esto abrirá tu aplicación en http://localhost:3000 y podrás ver los cambios en tiempo real.
