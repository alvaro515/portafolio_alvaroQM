Python y Flask

¿Qué es Python?

Python es un lenguaje de programación sencillo, poderoso y muy usado en backend, ciencia de datos e inteligencia artificial.

¿Qué es Flask?

Flask es un microframework de Python para crear aplicaciones web ligeras, rápidas y fáciles de entender.

Características de Flask

  • Minimalista y flexible
  • Ideal para APIs
  • Fácil integración con bases de datos
  • Perfecto para proyectos pequeños y medianos

Ejemplo Básico en Flask


from flask import Flask

app = Flask(__name__)

@app.route("/")
def inicio():
    return "Hola desde Flask"

if __name__ == "__main__":
    app.run(debug=True)

Flask y APIs

Flask se usa mucho para crear APIs REST que devuelven datos en formato JSON, ideales para conectar con frontend modernos.

Conclusión

Flask es como un backend zen: pequeño, claro y directo. Python piensa, Flask ejecuta.
