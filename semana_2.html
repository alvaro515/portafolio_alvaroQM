Teoría y Conceptos de la Semana

1. HTML5

HTML5 es la última versión del estándar HTML (HyperText Markup Language). Este lenguaje se utiliza para estructurar el contenido de las páginas web. Con HTML5, se han introducido una serie de mejoras que permiten la creación de aplicaciones web más potentes y con más funcionalidades.

¿Para qué sirve HTML5?

HTML5 se utiliza para estructurar y presentar contenido en la web. Permite agregar texto, imágenes, formularios, videos, audio, entre otros. Las nuevas etiquetas semánticas de HTML5 hacen que el contenido sea más accesible y más fácil de entender por los motores de búsqueda, lo que mejora el SEO (optimización para motores de búsqueda).

¿Cómo instalar HTML5?

HTML5 no necesita instalación, ya que es un estándar de marcado que se utiliza directamente en los archivos de tu proyecto. Simplemente debes asegurarte de que tu archivo tenga la extensión .html y que utilices las etiquetas adecuadas.

Ejemplo de Estructura Básica de HTML5:

        
          <!DOCTYPE html>
          <html lang="es">
            <head>
              <meta charset="UTF-8">
              <title>Mi Página Web</title>
            </head>
            <body>
              <h1>Bienvenido a mi página web</h1>
              <p>Este es un párrafo de ejemplo.</p>
            </body>
          </html>

Nuevas Etiquetas en HTML5

HTML5 introdujo una serie de nuevas etiquetas que mejoran la accesibilidad y la estructura semántica del contenido:

  • <header>: Define la cabecera de un documento o una sección.
  • <footer>: Define el pie de página de un documento o una sección.
  • <article>: Define un artículo independiente.
  • <section>: Define una sección de contenido.
  • <nav>: Define un bloque de navegación.
  • <aside>: Define contenido adicional, como una barra lateral.

2. XML

XML (eXtensible Markup Language) es un lenguaje de marcado que se utiliza para almacenar y transportar datos. A diferencia de HTML, XML no tiene etiquetas predefinidas y permite que los usuarios definan sus propias etiquetas para describir los datos.

¿Para qué sirve XML?

XML se utiliza principalmente para representar datos de manera estructurada. Es especialmente útil para intercambiar información entre sistemas diferentes. A diferencia de HTML, que está diseñado para mostrar contenido, XML solo describe los datos y su estructura.

¿Cómo instalar XML?

XML no requiere instalación. Es simplemente un formato de texto basado en etiquetas. Solo necesitas crear archivos con extensión .xml para empezar a trabajar con XML.

Ejemplo de Documento XML:

        
          <libros>
            <libro>
              <titulo>El Gran Libro</titulo>
              <autor>Juan Pérez</autor>
            </libro>
            <libro>
              <titulo>La Aventura de la Vida</titulo>
              <autor>María López</autor>
            </libro>
          </libros>

3. CSS3

CSS3 es la tecnología utilizada para diseñar la apariencia de las páginas web. Con CSS3, se pueden aplicar estilos como colores, fuentes, márgenes, bordes, y efectos de transición y animación, entre otros.

¿Para qué sirve CSS3?

CSS3 se utiliza para aplicar estilos a los elementos HTML de una página web. Permite modificar la disposición de los elementos en la página, su apariencia visual, y cómo interactúan con el usuario (por ejemplo, con animaciones).

¿Cómo instalar CSS3?

Al igual que con HTML5, **CSS3 no requiere instalación**. Solo necesitas crear un archivo con extensión .css y vincularlo en tu documento HTML utilizando la etiqueta <link>.

Ejemplo de cómo usar CSS3:

        
          /* Este es un archivo CSS */
          h1 {
            color: #ff6347; /* Color tomate */
            font-size: 3rem;
          }

          p {
            color: #4682b4; /* Color azul acero */
            font-size: 1.5rem;
          }

Flexbox y Grid Layout en CSS3

CSS3 introduce dos poderosas herramientas para crear diseños responsivos:

  • Flexbox: Permite alinear y distribuir elementos dentro de un contenedor de forma flexible, independientemente de su tamaño.
  • Grid Layout: Permite crear diseños de cuadrícula complejos con filas y columnas, haciendo que los diseños sean mucho más fáciles de manejar.

Ejemplo de Flexbox:

        
          .contenedor {
            display: flex;
            justify-content: space-between; /* Espaciado entre los elementos */
          }
          .elemento {
            width: 30%;
          }

Ejemplo de Grid Layout:

        
          .contenedor {
            display: grid;
            grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
          }
          .elemento {
            background-color: #f0f0f0;
            padding: 20px;
          }

4. Diseño Responsivo

El diseño responsivo asegura que las páginas web se vean bien en cualquier dispositivo, desde una computadora de escritorio hasta un teléfono móvil.

Para lograr un diseño responsivo, usamos **media queries** en CSS, que nos permiten aplicar estilos diferentes según el tamaño de la pantalla del dispositivo.

Ejemplo de Media Query:

        
          @media (max-width: 768px) {
            body {
              background-color: #f0f0f0;
            }
            h1 {
              font-size: 1.5rem;
            }
          }
