PHP y Laravel

¿Qué es PHP?

PHP es un lenguaje de programación del lado del servidor, muy usado para desarrollar aplicaciones web dinámicas.

¿Qué es Laravel?

Laravel es un framework de PHP que facilita el desarrollo backend usando el patrón MVC (Modelo – Vista – Controlador).

Ventajas de Laravel

  • Organización clara del código
  • Rutas simples y elegantes
  • ORM Eloquent para base de datos
  • Seguridad integrada

Estructura MVC


Modelo      → Maneja datos y base de datos
Vista       → Interfaz de usuario
Controlador → Lógica del negocio

Ejemplo de Ruta en Laravel


Route::get('/saludo', function () {
    return 'Hola desde Laravel';
});

Ejemplo de Controlador


class UsuarioController extends Controller {
    public function index() {
        return "Lista de usuarios";
    }
}

Conclusión

Laravel convierte el backend en una experiencia ordenada, segura y elegante. Es PHP, pero con traje y corbata.
