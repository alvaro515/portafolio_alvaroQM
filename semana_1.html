Teoría y Conceptos de la Semana

1. Soluciones Web

  • Sistema Web: Es una aplicación completamente accesible desde un navegador. A menudo interactúa con bases de datos y otros servicios a través de Internet. Ejemplo: Gmail (una aplicación de correo electrónico).
  • Aplicación Web: Son más complejas, permiten la interacción en tiempo real. Ejemplo: Google Docs (aplicación de edición de documentos en línea).
  • Sitio Web: Conjunto de páginas web interconectadas con contenido generalmente estático. Ejemplo: Blog personal o Sitio corporativo.
  • Página Web: Documento único accesible desde un navegador. Ejemplo: Página de contacto en un sitio web.

2. Tecnologías Web

HTML: Lenguaje de marcado para estructurar las páginas web. Definimos los elementos, como encabezados, párrafos, imágenes, enlaces y más.

Ejemplo de HTML:

        
          <html>
            <head><title>Mi Página</title></head>
            <body>
              <h1>Bienvenido a mi página</h1>
              <p>Este es un párrafo simple.</p>
            </body>
          </html>

CSS: Lenguaje de estilos que se usa para aplicar el diseño visual en las páginas HTML.

Ejemplo de CSS:

        
          h1 {
            color: blue;
            font-size: 2em;
          }
          p {
            color: gray;
            font-size: 1.2em;
          }

JavaScript: Lenguaje de programación que permite hacer las páginas web interactivas.

Ejemplo de JavaScript:

        
          function saludo() {
            alert("¡Hola, bienvenido!");
          }

3. Cómo Funciona la Web

DNS (Domain Name System): Convierte nombres de dominio (como google.com) en direcciones IP que las computadoras pueden entender.

Protocolo TCP/IP: Protocolo que permite que los datos viajen por Internet. Define cómo se transmiten los datos.

Protocolo HTTP/HTTPS: Protocolo utilizado para la comunicación entre clientes (navegadores) y servidores. HTTPS es la versión segura de HTTP.

4. Roles en el Desarrollo Web

  • Frontend: Desarrolla la parte visible e interactiva de la web.
  • Backend: Se encarga de la lógica del servidor y de la gestión de datos.
  • Fullstack: Se maneja tanto en frontend como en backend.

5. Herramienta de Desarrollo - Visual Studio Code

  • Visual Studio Code (VSC): Es un editor de código gratuito muy popular entre los desarrolladores.
  • Extensiones útiles: Prettier, Emmet, Live Server, etc.
  • Atajos de Teclado: Ctrl + Shift + P para abrir la paleta de comandos.
